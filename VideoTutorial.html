Before delve deeper into this,let's know about basics of HTML.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design webpages.
What Does HTML Do?
HTML is used to:
Structure the content of a webpage
Add headings, paragraphs, links, images, and more
Tell the browser how to display the content
